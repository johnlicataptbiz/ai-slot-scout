## Contributing

Quick guidelines to contribute to AI Slot Scout:

- Branch from `main` and open a PR for features/fixes.
- Keep TypeScript types strict; run `npm run typecheck` before opening PRs.
- Run unit tests and linters: `npm test` and `npm run lint`.
- Add tests for any behavior changes (unit or E2E).
- Follow Conventional Commits for commit messages.
